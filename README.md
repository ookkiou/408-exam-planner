---

## ✨ 这是什么？

一个基于 Claude 的 408 计算机考研智能规划助手。只需告诉它你的情况，就能获得：

- 📅 **个性化复习计划** —— 根据你的基础、时间、目标定制
- 📚 **知识点讲解** —— 用比喻帮你理解难懂的概念
- 🎯 **真题战术分析** —— 教你选择题排除法、大题解题思路
- 📊 **进度追踪** —— 每周检查，进度落后给你补救方案
- 💪 **算法题训练** —— 分类刷题清单，复杂度分析指导

### 🧠 用比喻讲知识点
&gt; **Cache 映射方式**就像书桌和书柜：
&gt; - 直接映射 = 每本书只能放固定位置
&gt; - 全相联 = 随便放
&gt; - 组相联 = 先分组，组内随便放

### 🎮 分析真题套路
- 选择题：排除法优先
- 算法题：先写伪代码再写代码
- PV 操作：先确定同步互斥关系
